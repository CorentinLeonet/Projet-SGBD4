# SGBD4 (EN)
This project is a simple order management application designed to learn how to use Spring Boot, Spring Data, Hibernate, and Thymeleaf.

## Technologies
- Java
- Spring Boot
- Spring Data
- Hibernate
- Thymeleaf

## Database Configuration
Make sure you have your MySQL database set up.  
You can modify the connection settings in the `application.properties` file.

## Running the Application
1. Open your preferred IDE (such as IntelliJ IDEA or Eclipse).
2. Run the main class `CorentinSgbd4Application.java`.  
   The application should launch on the default port (usually `localhost:8080`).

# SGBD4 (FR)
Ce projet est une application de gestion de commandes simple, dans le but d'apprendre à utiliser Spring Boot, Spring Data, Hibernate et Thymeleaf.

## Technologies
- Java
- Spring Boot
- Spring Data
- Hibernate
- Thymeleaf

## Configuration de la base de données
Assurez-vous que vous avez configuré votre base de données MySQL.  
Vous pouvez modifier les paramètres de connexion dans le fichier `application.properties`.

## Lancer l'application
1. Ouvrez votre IDE préféré (comme IntelliJ IDEA ou Eclipse).
2. Lancez la classe principale `CorentinSgbd4Application.java`.  
   L'application devrait se lancer sur le port par défaut (généralement `localhost:8080`).
